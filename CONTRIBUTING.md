# Contributing to Wingman

Thank you for your interest in contributing to Wingman! We welcome contributions of all kinds.

## How to Contribute

1. **Report Bugs:** Open an issue on GitHub.
2. **Suggest Features:** Open an issue on GitHub.
3. **Submit Pull Requests:**
   - Fork the repository.
   - Create a new branch for your feature or bug fix.
   - Write clean, documented code.
   - Ensure all changes align with the project's goal of being a "career wingman".
   - Open a PR against the `main` branch.

## Development Setup

- Follow the [README.md](README.md) instructions for setup.
- Use `npm run dev` to run with hot reloading.
- Adhere to the existing code style (ESM, modular architecture).

## Code of Conduct

Please be respectful and professional in all interactions within the community.
